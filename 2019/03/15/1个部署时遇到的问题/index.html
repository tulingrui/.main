1个困扰很久的问题

更新项目时很多时候只改了一部分功能
所以 更新时也是本地编译后更新修改的部分文件

本地研发环境运行时是毫无问题

但是更新到服务器上时偶尔就会报出

java (Ljava/lang/String;Ljava/lang/String;I)V

差不多就这种的 有斜杠斜杠分开的不同类型的错误

怀疑过是环境问题导致错误

但是我将所有文件一起发布时又没问题了

反复之后发现 将与更新文件有相关调用的文件一起发布才可以

但是不明白原因

找了很久 发现这篇博客

https://blog.csdn.net/dailywater/article/details/72848770

方法签名不一致
