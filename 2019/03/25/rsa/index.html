RSA加密导致内存溢出

RSA加密导致内存溢出

近期做的一个功能需要用到RSA加密

之前没接触过

搜了一下然后就开始做了 功能完成上线测试时就发现问题了

内存飙升且降不下来 直到内存溢出Tomcat自动关闭了

再搜了1下 好多人都遇到这个问题 原因就是 

BouncyCastleProvider

这个类 每次 new一个就多一个，不回收

所以内存见增不见减

所有new的地方改成调用 getInstance()

private static volatile BouncyCastleProvider bouncyCastleProvider = null;
public static BouncyCastleProvider getInstance() {
        if(null == bouncyCastleProvider){
            synchronized(RSAUtil.class){
                if(null == bouncyCastleProvider){
                    bouncyCastleProvider = new BouncyCastleProvider();
                }
            }
        }
        return bouncyCastleProvider;
 }
